# Experiment reports

Each report is a frozen write-up of one **experiment** — a self-contained run of the
research loop that produced one or more model versions. Reports are numbered and never
edited after the fact; new experiments get a new number. The living records are the
[leaderboard](../../projects/shakespeare/leaderboard.md), [MODELS.md](../../projects/shakespeare/MODELS.md), and the per-version
[model cards](../model-cards/).

| 02 | [Can you put an obsession on a dial?](experiment-02.md) — steerability + synthetic data, runs v1–v3 (bake the obsession into the corpus; the $0 ablation that found the dial's real bottleneck). | `→ gatsby-nanogpt-1` | Jun 2026 |

---

**To add a report:** copy the previous `experiment-NN.md` to the next number, write up
the experiment, add a row above, and link it from the version's model card. Don't edit a
published report — supersede it with a new one.

Charts are generated by the [`dataviz`](../../tools/dataviz/) pipeline (`uv run python tools/dataviz/build.py`)
and embedded as light/dark PNGs under [`assets/`](assets/).
